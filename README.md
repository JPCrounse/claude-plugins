# claude-plugins-jp

Personal Claude Code plugin marketplace.

## Installation

```bash
claude plugin install <plugin-name>@claude-plugins-jp
```

## Available Plugins

*No plugins yet — add your first plugin under `plugins/`.*

## Adding a Plugin

1. Create `plugins/<your-plugin>/`
2. Add `.claude-plugin/plugin.json` manifest
3. Add skills, agents, hooks, or commands
4. Register in `.claude-plugin/marketplace.json`
5. Validate with `claude plugin validate .`
